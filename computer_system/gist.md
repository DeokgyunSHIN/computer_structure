#  컴퓨터 시스템 과 구조

 > 컴퓨터 시스템은 하드웨어와 소프트 웨어로 구성되어져 있다.
 > 
 >  소프츠웨어 -> 운영체제와 응용 프로그램
 >  
 >  하드웨어 -> CPU, Memory, Storage, Network 등등 


### 컴퓨터 주요 구성 요소 (3가지)

> 1. CPU (중앙처리장치) 
> 
> 연산 (ALU)
> >
> > 산술 연산 -> 덧셈
> > 
> > 논리 연산  -> if 문이라고 생각하면 된다.
> 
> 제어 
> >
> > IO Device (입출력 장치), Memory, ALU 동작 제어

<br>
<br>
<br>

> 2. Memory( 데이터를 저장하는 장치)
> 
> 내부 기억장치 
> >
> > CPU 안에 레지스터, 캐쉬  (동작하기 위한 임시적으로 데이터를 저장할 수 있는 작은 공간)
> > 
> > DRAM등 메모리 (램)
> 
> 외부 기억장치 (보조기억장치)
> >
>  SSD, HDD
